EasyRecord个人数据采集

网站用于记录个人每天生活中随时产生的各种数据。主要包括以下功能：
    注册、登录：存在输入合法性验证。
    新建PDO模型：可以自定义PDO模型或选择公用的模板。
    添加PDO记录：选用所要使用的PDO模型，输入信息，保存记录。
    查找PDO记录：选择PDO名称，输入关键字进行查询。
    聚合搜索：通过输入想要查询的内容的部分字段，搜索出包含该关键字的记录，可以实现对数据的任何维度的查询。
    Excel文件导入：可以导入Excel格式的个人数据到某个特定的PDO，或者根据Excel的表头自动生成新的PDO。
